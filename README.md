# EduTrack Landing Page

## Overview
This project is a landing page designed for school principals and academic heads who need quick, trustworthy insight into student performance analytics.

## Problems Identified
- School leaders are busy and need fast clarity.
- Academic data is scattered across multiple reports.
- Existing tools are complex and not decision-focused.

## Priorities Chosen
- Clear messaging within first screen.
- Trust-focused design with simple metrics.
- Minimal navigation to reduce cognitive load.
- Dashboard preview to quickly explain product value.

## What I Would Do Next

- Include testimonials from schools.
- Implement analytics tracking.
- Improve accessibility and mobile optimization.
